## [1.0.0] — 2026-04-13

Первый стабильный релиз. Публичный API зафиксирован.

### Реализовано

- **`gost341112`** — Стрибог (ГОСТ Р 34.11-2012), 256 и 512 бит, интерфейс `hash.Hash`. KAT: RFC 6986, Приложение А.
- **`gost341215`** — Кузнечик и Магма (ГОСТ Р 34.12-2015), интерфейс `cipher.Block`. Precomputed S+L таблицы, нулевые аллокации. KAT: RFC 7801, RFC 8891.
- **`gost341315`** — режимы ECB/CBC/CFB/CTR/OFB/MAC (ГОСТ Р 34.13-2015) с поддержкой GOST shift-register (m > n). KAT: Приложение А стандарта.
- **`gost341012`** — ЭЦП (ГОСТ Р 34.10-2012), 7 стандартных кривых (256 и 512 бит), `crypto.Signer`. KAT: RFC 7091.
- **`mgm`** — AEAD Multilinear Galois Mode (RFC 9058), интерфейс `cipher.AEAD`. KAT: Приложение A.
- **`vko`** — ключевое согласование (RFC 7836), 256/512-бит кривые. KAT: Приложение B.
- **`kdf`** — HMAC/CMAC/KDF/PBKDF2/key wrap (Р 50.1.113-2016, Р 50.1.111-2016).
- **`x509gost`** — маршалинг ключей ГОСТ в форматы PKIX / PKCS#8.

### Свойства

- Нулевые внешние зависимости — только стандартная библиотека.
- Constant-time операции на чувствительных путях.
- Обнуление ключевого материала после использования.
- CI: Go 1.23 и 1.24, `go test`, `go vet`, бенчмарки.
