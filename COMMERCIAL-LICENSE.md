# Коммерческая лицензия gost-crypto

## Зачем нужна

gost-crypto распространяется под AGPL-3.0. Это означает, что любое ПО,
использующее gost-crypto и распространяемое третьим лицам (включая SaaS),
должно быть опубликовано под AGPL-3.0 с полным исходным кодом.

Коммерческая лицензия снимает это ограничение.

## Что даёт коммерческая лицензия

- Использование gost-crypto в закрытом коммерческом ПО
- Распространение без раскрытия исходного кода
- SaaS/облачные сервисы без ограничений AGPL
- Приоритетная поддержка
- Возможность модификации без обязательства публикации изменений

## Для кого

- Коммерческие компании, распространяющие ПО с ГОСТ-криптографией
- SaaS-провайдеры, использующие ГОСТ-алгоритмы
- Системные интеграторы, встраивающие ГОСТ в клиентские решения
- Государственные подрядчики с требованиями закрытого кода

## Кому коммерческая лицензия НЕ нужна

- Open-source проекты (AGPL-совместимые)
- Внутреннее использование без распространения
- Исследования и образование
- Личные проекты
